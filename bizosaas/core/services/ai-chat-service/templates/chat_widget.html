Integration Code

Add this to your admin dashboard to embed the AI chat widget:

<!-- Include the chat widget script -->
<script src="http://localhost:3003/chat/embed.js"></script>

<script>
// Initialize chat widget
const chat = new BizOSaaSChat({
    apiUrl: 'http://localhost:3003',
    authToken: 'your-jwt-token-here',
    position: 'bottom-right', // bottom-right, bottom-left, top-right, top-left
    container: 'custom-chat-container' // optional
});
</script>

API Endpoints

POST /chat/message          - Send chat message
GET  /chat/assistants       - Get available assistants
GET  /chat/history/{id}     - Get chat history
DELETE /chat/session/{id}   - Delete chat session
WS   /chat/ws/{session_id}  - WebSocket connection

Dashboard Integration Examples

TailAdmin Dashboard (localhost:3001)

// Add to your TailAdmin layout
import { useAuth } from './hooks/useAuth';

const { token } = useAuth();

useEffect(() => {
    if (token) {
        const chat = new BizOSaaSChat({
            apiUrl: 'http://localhost:3003',
            authToken: token,
            position: 'bottom-right'
        });
    }
}, [token]);

SQLAdmin Dashboard (localhost:5000)

// Add to your Flask/FastAPI template
<script>
// Get auth token from session or API
fetch('/api/auth/token')
    .then(response => response.json())
    .then(data => {
        const chat = new BizOSaaSChat({
            apiUrl: 'http://localhost:3003',
            authToken: data.token,
            position: 'bottom-left'
        });
    });
</script>
